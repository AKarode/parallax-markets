## What Was Built

### WorldState.copy() (world_state.py)
- Added `copy.deepcopy`-based `copy()` method to WorldState class
- Enables safe mutation of world state during cascade operations without corrupting shared state
- Tests prove mutation isolation: modifying copy does not affect original

### PredictionOutput Schema Extension (schemas.py)
- Added 3 optional fields: `ensemble_probabilities`, `ensemble_std_dev`, `ensemble_is_unstable`
- Backward compatible via `| None = None` defaults (same pattern as existing optional fields)

### Ensemble Prediction Utility (ensemble.py)
- `ensemble_predict()`: Makes 3 concurrent Claude calls via `asyncio.gather` at temperatures 0.3, 0.5, 0.7
- `trimmed_mean()`: Drops min/max from 3 values (equals median); falls back to arithmetic mean for n<=2
- `compute_ensemble()`: Aggregates probabilities, computes std_dev, flags instability when std_dev > 0.10
- `parse_llm_json()`: Extracts JSON from markdown-fenced LLM responses (consolidates duplicated logic from 3 predictors)
- Budget records each of 3 calls individually (not 1 batch record)
- Partial failures handled gracefully: if 1 of 3 calls fails to parse, ensemble continues with 2

### Tests
- 7 tests in test_world_state.py (2 new: copy independence, tick preservation)
- 11 tests in test_ensemble.py covering all functions including async ensemble_predict with partial failures
