1. Create `backend/src/parallax/prediction/ensemble.py` with these functions:

   ```python
   """Ensemble prediction utility -- multi-call LLM with aggregation."""
   from __future__ import annotations

   import asyncio
   import json
   import logging
   import statistics
   from typing import Any

   logger = logging.getLogger(__name__)

   ENSEMBLE_TEMPERATURES = [0.3, 0.5, 0.7]
   INSTABILITY_THRESHOLD = 0.10  # 10 percentage points
   ```

   a. `parse_llm_json(raw_text: str) -> dict`:
      - Strips markdown code fences (```json ... ```) using the exact pattern from oil_price.py lines 139-152.
      - Calls `json.loads()` on cleaned text.
      - Raises `ValueError` on parse failure with first 200 chars of text.
      - This extracts the duplicated parse logic from all 3 predictors.

   b. `trimmed_mean(values: list[float]) -> float`:
      - If len <= 2, return `statistics.mean(values)`.
      - Otherwise, sort, drop first and last, return `statistics.mean` of remainder.
      - With n=3 this equals median. Document this in docstring.

   c. `compute_ensemble(probabilities: list[float]) -> dict`:
      - Returns dict with keys: `probability` (float), `std_dev` (float), `is_unstable` (bool), `individual_probabilities` (list[float]).
      - If len < 2: probability = first value (or 0.5 if empty), std_dev = 0.0, is_unstable = False.
      - Otherwise: probability = `trimmed_mean(probabilities)`, std_dev = `statistics.stdev(probabilities)`, is_unstable = std_dev > INSTABILITY_THRESHOLD.

   d. `async def ensemble_predict(client: Any, model: str, prompt: str, budget: Any, max_tokens: int = 2000) -> dict`:
      - Makes 3 concurrent calls via `asyncio.gather` with temperatures from ENSEMBLE_TEMPERATURES.
      - Each call: `client.messages.create(model=model, max_tokens=max_tokens, temperature=temp, messages=[{"role": "user", "content": prompt}])`
      - For each response, records budget: `budget.record(resp.usage.input_tokens, resp.usage.output_tokens, "sonnet")` -- note "sonnet" not "opus" per research recommendation.
      - Parses each response with `parse_llm_json()`. If a parse fails, logs warning and skips that response (does not crash).
      - Collects probabilities from parsed["probability"] for each successful parse.
      - Calls `compute_ensemble(probabilities)`.
      - Picks the median response's full parsed dict for reasoning/evidence (the one closest to the ensemble probability).
      - Returns dict: `{"ensemble": ensemble_result, "parsed": median_parsed, "all_parsed": all_parsed_list, "call_count": len(successful_parses)}`.
      - If ALL 3 calls fail to parse, raises `ValueError("All ensemble calls failed to parse")`.
