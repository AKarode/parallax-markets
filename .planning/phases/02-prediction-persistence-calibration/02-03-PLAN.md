    1. `def hit_rate_by_proxy_class(conn: duckdb.DuckDBPyConnection) -> list[dict]`:
       ```sql
       SELECT
           proxy_class,
           COUNT(*) AS total,
           SUM(CASE WHEN model_was_correct THEN 1 ELSE 0 END) AS correct,
           ROUND(SUM(CASE WHEN model_was_correct THEN 1 ELSE 0 END)::DOUBLE / COUNT(*), 3) AS hit_rate
       FROM signal_ledger
       WHERE model_was_correct IS NOT NULL
       GROUP BY proxy_class
       ORDER BY proxy_class
       ```
       Return list of dicts: [{"proxy_class": str, "total": int, "correct": int, "hit_rate": float}]

    2. `def calibration_curve(conn: duckdb.DuckDBPyConnection) -> list[dict]`:
       ```sql
       SELECT
           CASE
               WHEN model_probability < 0.2 THEN '0-20%'
               WHEN model_probability < 0.4 THEN '20-40%'
               WHEN model_probability < 0.6 THEN '40-60%'
               WHEN model_probability < 0.8 THEN '60-80%'
               ELSE '80-100%'
           END AS bucket,
           COUNT(*) AS n,
           ROUND(AVG(model_probability), 3) AS avg_predicted,
           ROUND(AVG(CASE WHEN model_was_correct THEN 1.0 ELSE 0.0 END), 3) AS actual_rate
       FROM signal_ledger
       WHERE model_was_correct IS NOT NULL
       GROUP BY bucket
       ORDER BY bucket
       ```
       Return list of dicts: [{"bucket": str, "n": int, "avg_predicted": float, "actual_rate": float}]

    3. `def edge_decay(conn: duckdb.DuckDBPyConnection) -> list[dict]`:
       ```sql
       SELECT
           CASE
               WHEN ABS(effective_edge) < 0.05 THEN '<5%'
               WHEN ABS(effective_edge) < 0.10 THEN '5-10%'
               WHEN ABS(effective_edge) < 0.15 THEN '10-15%'
               ELSE '15%+'
           END AS edge_bucket,
           COUNT(*) AS n,
           ROUND(AVG(ABS(effective_edge)), 3) AS avg_edge,
           ROUND(AVG(realized_pnl), 4) AS avg_pnl,
           ROUND(AVG(CASE WHEN model_was_correct THEN 1.0 ELSE 0.0 END), 3) AS hit_rate
       FROM signal_ledger
       WHERE realized_pnl IS NOT NULL
       GROUP BY edge_bucket
       ORDER BY edge_bucket
       ```
       Return list of dicts: [{"edge_bucket": str, "n": int, "avg_edge": float, "avg_pnl": float, "hit_rate": float}]

    4. `def _check_minimum_data(conn: duckdb.DuckDBPyConnection, min_days: int = 7) -> tuple[bool, str]`:
       - Query: `SELECT MIN(created_at) AS first, MAX(created_at) AS last, COUNT(*) AS total FROM prediction_log`
       - If no rows: return (False, "No prediction data found")
       - Compute days_span = (last - first).days
       - If days_span < min_days: return (False, f"Insufficient data: {days_span} days of predictions (minimum {min_days} required)")
       - Else: return (True, f"{total} predictions over {days_span} days")
       Per PERS-04 and D-07.

    5. `def calibration_report(conn: duckdb.DuckDBPyConnection) -> str`:
       - Call _check_minimum_data(). If insufficient, return the insufficient message string (don't raise -- per CLI-friendly design).
       - Format text report with sections:
         ```
         === PARALLAX CALIBRATION REPORT ===
         Data: {total} predictions over {days} days

         --- HIT RATE BY PROXY CLASS ---
         {proxy_class:<15} {total:>5} {correct:>7} {hit_rate:>8}

         --- CALIBRATION CURVE ---
         {bucket:<10} {n:>5} {avg_predicted:>12} {actual_rate:>11}

         --- EDGE DECAY ---
         {edge_bucket:<10} {n:>5} {avg_edge:>9} {avg_pnl:>9} {hit_rate:>8}
         ```
       - If any section has no data (all queries empty), print "No resolved signals yet." under that section.

    All queries use literal SQL (no user input, per D-05 tiny data volume). All return list[dict] for easy formatting and testing.
