class Divergence(BaseModel):
    model_id: str
    prediction: PredictionOutput
    market_price: MarketPrice
    model_probability: float
    market_probability: float
    edge: float  # model_prob - market_prob (positive = model thinks underpriced)
    edge_pct: float  # edge as percentage
    signal: str  # "BUY_YES", "BUY_NO", "HOLD"
    strength: str  # "strong" (>15%), "moderate" (10-15%), "weak" (5-10%)
    created_at: datetime

class DivergenceDetector:
    def __init__(self, min_edge_pct: float = 5.0):
        self._min_edge = min_edge_pct / 100.0

    def detect(
        self,
        predictions: list[PredictionOutput],
        market_prices: list[MarketPrice],
    ) -> list[Divergence]:
        """Compare model predictions against market prices. Flag divergences above threshold.

        Matching logic: Match predictions to markets by prediction_type + ticker mapping.
        A prediction with kalshi_ticker or polymarket_id gets matched to the corresponding MarketPrice.

        Signal logic:
        - If model_prob > market_prob + min_edge: BUY_YES (model thinks event more likely than market)
        - If model_prob < market_prob - min_edge: BUY_NO (model thinks event less likely)
        - Otherwise: HOLD

        Strength:
        - |edge| > 0.15: "strong"
        - |edge| > 0.10: "moderate"
        - |edge| > 0.05: "weak"
        """
```

**Pipeline orchestration in `async def run_brief(dry_run: bool = False, no_trade: bool = False)`:**

1. **Initialize clients:**
   - KalshiClient (from env: KALSHI_API_KEY, KALSHI_PRIVATE_KEY_PATH)
   - PolymarketClient (no auth)
   - Anthropic client (from env: ANTHROPIC_API_KEY)
   - BudgetTracker(daily_cap_usd=20.0)
   - Load ScenarioConfig from scenario_hormuz.yaml
   - Create CascadeEngine(config)

2. **Fetch data (parallel via asyncio.gather):**
   - GDELT: Use volume_gate + structural_dedup on recent events (if BigQuery available). Fallback: skip GDELT if no credentials, use empty events list with warning.
   - EIA: fetch_brent(api_key=os.environ.get("EIA_API_KEY", ""))
   - Kalshi: Search for Hormuz, ceasefire, oil markets
   - Polymarket: get_iran_markets()

3. **Run predictions (parallel via asyncio.gather):**
   - OilPricePredictor.predict()
   - CeasefirePredictor.predict()
   - HormuzReopeningPredictor.predict()

4. **Detect divergences:**
   - DivergenceDetector.detect(predictions, market_prices)

5. **Output brief to stdout:**
   Format as structured text:
   ```
   ============================================
   PARALLAX DAILY INTELLIGENCE BRIEF
   {date} | Budget: ${spent:.2f}/${cap:.2f}
   ============================================

   --- PREDICTIONS ---

   OIL PRICE (Brent)
   Direction: {direction} | Magnitude: ${low}-${high}
   Confidence: {confidence:.0%} | Timeframe: {timeframe}
   Reasoning: {reasoning}

   CEASEFIRE PROBABILITY
   Probability: {prob:.0%} | Timeframe: {timeframe}
   Confidence: {confidence:.0%}
   Reasoning: {reasoning}

   HORMUZ REOPENING
   Probability: {prob:.0%} | Timeframe: {timeframe}
   Confidence: {confidence:.0%}
   Reasoning: {reasoning}

   --- MARKET PRICES ---

   {table of Kalshi + Polymarket prices}

   --- DIVERGENCES ---

   {for each divergence with edge > 5%:}
   SIGNAL: {BUY_YES/BUY_NO} {ticker}
   Model: {prob:.0%} vs Market: {market_prob:.0%} | Edge: {edge:.1%} ({strength})
